# SWOT 引导对话示例（Agent ↔ 用户）

> 本文件提供**真实对话示范**，供 Agent 学习"怎么引导、怎么追问、怎么守住不替写"。
> 所有「用户」发言为模拟，旨在演示交互节奏；Agent 的每一句都遵循 interaction_guide.md 的纪律。

---

## 示例一 · 门1 优势（S）完整引导

**场景**：分析对象 = 某汽车零部件供应商品质部，用户是品质总监。

**Agent**：我们先梳理「优势（Strengths）」。优势是组织内部、当前已经具备的积极条件。您可以从这几个角度想：团队能力、管理体系、技术专长、设备/资源、数据能力、客户关系、质量管控。
举个例子（仅为示范，不代表您的真实情况）："质量体系健全、检测设备齐全、客户认可度高"。
请结合「品质部」的实际情况，告诉我您认为它最强的 3-5 条优势是什么？

**用户**：我们体系是有的，IATF16949 前年过的。

**Agent**：好的，IATF16949 认证通过是一个实打实的管理体系优势 👍 我帮您记下来：
- 已通过 IATF16949 体系认证（2024 年）
除了体系，在「检测设备 / 数据能力 / 团队」这几个角度，还有没有能算优势的？比如有没有哪类检测设备是同行少有的，或者团队里有没有特别稳的核心骨干？

**用户**：检测设备还行吧，三坐标、X 光都有。就是人流动有点大。

**Agent**：明白，我这样记：
- 检测设备较全（三坐标、X 光等关键设备齐全）
您提到「人流动大」——这其实更偏劣势（内部消极因素），我们到门2 会专门聊，先不急着归到这里。
那团队能力上，有没有正向的点？比如核心岗稳定性、或者某类技能特别强？

**用户**：哦对，我们几个老师傅 SPC 用得特别溜，客户审核基本没在统计过程这块栽过。

**Agent**：这条很好，记下来：
- 核心骨干 SPC 应用能力强，客户审核中统计过程控制表现稳定
目前优势我们有 3 条（IATF16949 认证 / 检测设备齐全 / SPC 核心能力）。按您的判断，再加 1-2 条能不能凑到 3-5 条？或者这 3 条已经是您最看重的了？

**用户**：就这 3 条吧，够了。

**Agent**：收到，门1 优势锁定为：
1. 已通过 IATF16949 体系认证（2024 年）
2. 检测设备较全（三坐标、X 光等关键设备齐全）
3. 核心骨干 SPC 应用能力强，客户审核中统计过程控制表现稳定
接下来进**门2 劣势（Weaknesses）**，我们聊聊内部当前不足的地方……

> **示范要点**：Agent 把用户零散的话**复述确认**成结构化条目（不擅自改写原意），遇到"人流动大"主动提示"这更像劣势、留到门2"，遇到含糊的"还行吧"用具体角度追问，全程**不替用户下结论**。

## 示例二 · 守住边界 + 用户跳过某门

**场景**：聊到门3 机会（O），用户想让 Agent 直接编。

**用户**：机会你帮我写几条吧，你们做质量的肯定比我知道。

**Agent**：我可以给您格式和思考角度，但具体每条得由您定——因为您比我最了解贵司实际面临的市场和政策环境 😊
机会（外部利好）一般从这几个角度找：市场需求、政策支持、技术趋势、客户需求、行业标准、合作资源。
比如（仅为示范）："客户质量要求提升、精益管理导入、行业政策利好"。
您先挑一个最明显的外部利好说说？哪怕一条我们也先记上。

**用户**：行吧……那客户今年确实提了零缺陷要求。

**Agent**：这条很具体，记下来：
- 客户提出零缺陷交付要求（机会：倒逼能力升级的外部拉力）
还有没有类似的？比如政策或技术侧有没有能借力的？

**用户**：其他的我一时想不起来，跳过吧。

**Agent**：理解，门3 机会目前只有 1 条（客户零缺陷要求）。
⚠️ 提示：机会象限条目偏少，会影响后面 **SO 战略组合**（优势×机会）的生成质量——SO 是"用优势抓机会"，机会少了 SO 就单薄。
我先按「**供参考·待确认**」标注这个象限为待补充，您后面想到随时补；不阻塞流程，我们继续门4 威胁。

> **示范要点**：用户想甩手时，Agent **不接锅**、给框架不给答案；用户跳过时用「供参考·待确认」+ 明确告知"会影响 SO 组合失真"，既尊重用户主导权，又把风险说清楚。
